Piet Mondrain's famous Composition with Red, Blue and Yellow (1930) is a great candidate to recreate using CSS Grid. The geometric artwork lends itself to arranging blocks of color in defined positions on a 2D grid. CSS Grid is designed for exactly that. You can define the block areas in just a few lines. In the artwork above there are seven blocks of color. Defining how those seven blocks should appear relative to each other in a grid is defined in this single line: grid-template-areas: "a a b b b b" "c c b b b b" "c c b b b b" "d d e e e f" "d d e e e g";.

That line of code may make a bit more sense when viewed this way:
grid-template-areas:
"a a b b b b"
"c c b b b b"
"c c b b b b"
"d d e e e f"
"d d e e e g";

Each line above (enclosed in quotes) represents a row of the grid. Once each area is defined, it's simple to tell which block belongs in which area. That would look like: .item2 { grid-area: b;}. A great, in-depth explanation of CSS Grid template areas from Smashing Magazine is a great resource.
