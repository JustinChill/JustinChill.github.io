Tic-Tac-Toe

A classic game rendered entirely with CSS selectors and state logic

Player Computer

The game of Tic-Tac-Toe is played on a 3×3 grid. This version uses zero JavaScript for game logic—instead, it relies on hidden radio inputs, CSS :checked pseudo-classes, and sibling combinators to calculate computer turns and detect winning states.
