Memes are for everybody, and making them accessible on the web is so easy! The memes on this page are all made using regular HTML & CSS, using the power of CSS to format everything. It's accessible to screenreaders (offering up a description each image) and can be easily changed. Inspiration for this website from Adrian Roselli.
