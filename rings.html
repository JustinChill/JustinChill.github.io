Spinning Rings Illusion

This clever visualization is made by stacking circles one on top of another and spinning them about their origin at different speeds. Each circle spins at a different rate relative to it's order in the pile. A CSS keyframe animation is used to transform each circle around its origin. Below is a single circle, following the same basic animation loop. Notice the patterns of overlap within the spinning rings- this is done using rgba colors with transparency.

Each circle's animation is staggered according to its position within the order. Doing so allows the circles to overlap each other as they rotate. Original animation is from Miriam Nadler!

SVGs are so great. Consider the following static circle. It is created using just a single SVG image and the inset shadow property for the depth effect. Inspiration for this illustration from Matys.
