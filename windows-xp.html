Windows XP: A love story

May 2021

Windows XP introduced a suite of features that made using a computer accessible, including a refreshed user interface and faster user switching. When the operating system was released in 2001, it was many people's first experience with a home computer.

Around the world, 140 million computers still run Microsoft XP, including millions of ATMs. As of May 2021, 0.7% of Windows PCs worldwide run Windows XP (4% in Africa), and 0.23% of all devices across all platforms run Windows XP. Microsoft stopped offering support for the operating system in 2014.

The loading animation is pretty simple to recreate. Three p elements are used for the text, and the boxes in the progress bar are rendered using a linear gradient.
