Mysteries

This is comic #1501- "Mysteries", enhanced by Justin Chill. Each mystery on the XY chart above links to the Wikipedia page for more information. This is accomplished using an HTML image map, with which you define clickable areas of a given image. In the example of the Mysteries comic above each mystery is defined as an area within given coordinates. An example would look like: coords="358,393,421,393,385,440,354,422" for the area "Bigfoot".
