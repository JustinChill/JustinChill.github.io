Xkcd is my favorite web comic. This post is really just an excuse to use this font I stumbled across recently. It recreates the handwriting style of XKCD creator Randall Munroe. Here I've used it to recreate comic #1820 - Security Tips. All credit to Randall for the inspiration!

Mysteries

This is comic #1501- "Mysteries". A layer on top of the comic links to the Wikipedia page of each dot. Fortunately, HTML provides us with a built in function for making this layer. It's called an HTML image <map>. An image map is an image with clickable areas. The areas are defined with one or more <area> tags.
